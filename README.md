# fermi-estimation-animal-advocacy
Streamlit demo of the Fermi-estimation concept. 

I used the streamlit web app to to calculate the total number of birds inside an industrial agricultural facility using Fermi estimation. Because exact numbers are hidden by the industry, this tool breaks down the massive, unobservable total into small, measurable variables (like building size, number of floors, and space per bird) to generate a mathematically sound estimate.
